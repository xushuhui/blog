## 前言

当我们使用 MySQL 的时候，它对我们而言就是个黑盒。它的运行原理是怎样的？它是怎样执行 SQL 语句 ？它怎么把数据查询出来 / 写入进去？今天我们就以一条简单的查询 SQL 为
例回答这些问题。

### MySQL 基本架构

我们先来看这条 SQL，相信大家都看的懂，我们所写的代码中涉及到 MySQL 操作时都要用这条 SQL，可能有的是 ORM 已封装好了，但是不论哪种编程语言，发送给 MySQL 的 SQL 语句一定
是标准格式。

```sql
mysql> select * from user_info where ID=10；
```
